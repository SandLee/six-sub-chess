# six-sub-chess
cocos2d-x 3.0实现的六子棋游戏。这是一种民间流传的棋类游戏，小时候玩过，故而将其实现以重温旧梦，现将其开源，也算为非物质文化遗产保护做贡献，哈哈~

游戏玩法规则非常简单，就是两个吃一个。目前已有机器人陪玩功能，以后可能会加入双人对战和局域网联机对战的玩法。
